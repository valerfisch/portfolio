### Project Overview
A multi-platform 3D progressive web app for communicating BMW Group's brand behavior guidelines. The goal was to replace static brand books with something interactive that employees would actually want to use, while keeping it consistent across BMW and MINI.

### Challenge
BMW Group needed a secure, accessible tool to standardize brand behavior knowledge across the organization. It had to work on any device, support multiple languages, and feel like a product worth the brand name.

### My Role
As **Lead Full-Stack Developer and UI Designer**, I architected the application from the ground up. I built the responsive frontend with Vue and WebGL/Three.js, developed the gamified learning modules, and implemented the backend services for multilingual content delivery. The platform runs on smartphones, tablets, and desktops.

### Solution
A 3D progressive web app with multilingual strategy content, interactive learning modules, real-world guideline applications, rich media, and gamification elements for engagement and knowledge retention.

**Tech:** Vue, WebGL, Three.js, Node.js, REST APIs, Progressive Web App
